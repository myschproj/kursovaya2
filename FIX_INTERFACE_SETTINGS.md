# Исправление настроек интерфейса

В этой версии исправлено применение настроек интерфейса:

- тема `light`, `dark`, `contrast` теперь меняется через глобальную тему Vuetify;
- размер текста применяется ко всем основным компонентам;
- режим доступности увеличивает высоту элементов управления и межбуквенный интервал;
- настройки загружаются сразу после входа пользователя в систему;
- после нажатия «Сохранить настройки» изменения применяются без перезагрузки страницы.

Если браузер показывает старую версию, нажмите `Ctrl + F5` или перезапустите frontend:

```powershell
cd frontend
npm run dev
```
